# 코딩 연습 - 익명 타입

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 익명 타입 생성

`new { }` 구문을 사용하여 이름, 학년, 점수를 가진 익명 타입 객체를 생성하고, 각 속성을 출력하세요.

**실행 결과**

```
이름: 김철수
학년: 3
점수: 95.5
```

---

## 2. 기존 변수를 사용한 속성 이름 추론

### 2-1. 명시적 이름 지정

문자열 변수 `name`과 정수 변수 `age`를 선언하고, `new { Name = name, Age = age }` 형태로 명시적 이름을 지정하여 익명 타입을 생성하고 출력하세요.

**실행 결과**

```
person1: 이영희, 28
```

### 2-2. 변수명으로 자동 추론

같은 변수를 `new { name, age }` 형태로 사용하여 변수명이 자동으로 속성 이름이 되는 것을 확인하세요.

**실행 결과**

```
person2: 이영희, 28
```

---

## 3. 표현식에서 속성 이름 추론

문자열 변수와 해당 문자열의 `Length` 속성을 익명 타입에 넣어, 프로퍼티 이름이 자동으로 추론되는 것을 확인하세요.

**실행 결과**

```
텍스트: Hello
길이: 5
```

---

## 4. 복합 속성 이름 추론 예제

변수명 그대로 추론하는 방식과 명시적 이름을 지정하는 방식을 혼합하여 상품 정보 익명 타입을 생성하고 출력하세요. 가격은 천 단위 구분 기호를 포함하세요.

**실행 결과**

```
ID: 100
상품명: 마우스
가격: 25000
포맷된 가격: 25,000원
```

---

## 5. 타입 동일성

같은 속성 구조를 가진 두 익명 타입이 같은 타입인지 `GetType()` 비교로 확인하고, 할당이 가능한지 테스트하세요.

**실행 결과**

```
같은 타입: True
point1: (30, 40)
```

---

## 6. 값 동등성 (Equals)

같은 값과 다른 값을 가진 익명 타입 객체를 비교하여 `Equals()` 메서드의 구조적 동등성을 확인하세요.

**실행 결과**

```
a.Equals(b): True
a.Equals(c): False
```

---

## 7. 참조 동등성 (==)

`==` 연산자가 익명 타입에서 참조 비교를 수행하는 것을 확인하세요. 같은 값의 서로 다른 객체와, 같은 참조를 할당한 변수의 비교 결과를 출력하세요.

**실행 결과**

```
a == b: False
a == c: True
```

---

## 8. 익명 타입 배열

같은 구조의 익명 타입 객체 3개로 배열을 생성하고, `foreach`로 순회하며 출력하세요.

**실행 결과**

```
=== 학생 점수 목록 ===
김철수: 85점
이영희: 92점
박민수: 78점
```

---

## 9. 배열 요소 접근

익명 타입 배열에서 인덱스로 특정 요소에 접근하고, 배열 길이와 전체 가격 합계를 계산하여 출력하세요.

**실행 결과**

```
첫 번째 상품: 키보드
상품 개수: 3
총 가격: 375,000원
```

---

## 10. 구조 기반 타입 동일성

같은 속성 구조의 익명 타입 객체를 생성한 후, 같은 변수에 다른 값의 동일 구조 객체를 재할당하여 구조적 타입 동일성을 확인하세요.

**실행 결과**

```
1 - Object 1
2 - Object 2
```

---

## 11. 익명 타입 vs 튜플

동일한 데이터를 익명 타입과 튜플로 각각 생성하고, 사용 방식이 유사함을 확인하세요.

**실행 결과**

```
익명 타입: 홍길동, 30
튜플: 홍길동, 30
```

---

## 12. ToString() 메서드

익명 타입의 `ToString()` 메서드가 자동으로 구현되어 속성 정보를 문자열로 반환하는 것을 확인하세요. `ToString()` 호출과 직접 출력 두 가지 방법을 모두 테스트하세요.

**실행 결과**

```
{ Name = 김영희, Age = 25, City = 서울 }
{ Name = 김영희, Age = 25, City = 서울 }
```

## 13. 데이터 그룹화

직원 정보(이름, 부서, 급여)를 익명 타입 배열로 생성하고, `foreach`로 순회하며 출력하세요. 급여는 천 단위 구분 기호를 포함하세요.

**실행 결과**

```
=== 직원 정보 ===
김철수 (개발팀) - 5,000만원
이영희 (기획팀) - 4,500만원
박민수 (개발팀) - 5,500만원
```

두 개의 배열(이름 배열, 가격 배열)의 데이터를 결합하여 익명 타입 배열로 만들고 출력하세요.
