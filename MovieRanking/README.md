# MovieRanking

영화 정보를 익명 타입으로 생성하고 출력하는 프로그램을 작성하세요.

## 요구사항

- 다음 변수를 먼저 선언
  - `title`: "인터스텔라"
  - `genre`: "SF"
  - `rating`: 9.2
  - `releaseYear`: 2014
- 위 변수를 사용하여 **속성 이름 추론**으로 익명 타입 객체를 생성
- 각 정보를 형식에 맞게 출력
- 익명 타입의 `ToString()` 결과도 출력

## 예상 실행 결과

```
=== 영화 정보 ===
제목: 인터스텔라
장르: SF
평점: 9.2
개봉연도: 2014년

ToString 결과:
{ title = 인터스텔라, genre = SF, rating = 9.2, releaseYear = 2014 }
```
